# README

# SIMPLE RAILS 6 REST API

REST API to perform CRUD functions. Uses SQLite database by default.

## RUBY Version

    ruby 2.6.5p114 (2019-10-01 revision 67812)
    Rails 6.0.2.1

## INSTALL DEPENDENCIES
```bash
$ bundle install
```

## DB RUN MIGRATION
```bash
$ rails db:migrate
```

## RUN SERVER
```bash
$ rails s
```
